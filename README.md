# To-Do List

Aplicação web simples para gerenciamento de tarefas.

Este projeto é uma **lista de tarefas (To-Do List)** desenvolvida para fins de aprendizado em **desenvolvimento front-end**, permitindo criar, visualizar e remover tarefas diretamente na interface.

O objetivo foi praticar **JavaScript básico**, manipulação do **DOM**, interação com formulários e organização de interface.

---

## Sobre o projeto

O site funciona como um gerenciador simples de tarefas.  
O usuário pode adicionar novos itens à lista, visualizar as tarefas criadas e removê-las quando desejar.

A aplicação foi pensada para ser **simples, direta e funcional**, focando no aprendizado das interações básicas entre **HTML, CSS e JavaScript**.

---

## Funcionalidades

A aplicação possui as seguintes funcionalidades:

### Criar item
O usuário pode adicionar uma nova tarefa digitando o nome do item.

### Deletar item
Cada tarefa possui um botão que permite removê-la individualmente da lista.

### Deletar todos os itens
Existe uma opção para **remover todas as tarefas da lista de uma vez**.

### Scroll automático
Quando a quantidade de tarefas ultrapassa o espaço disponível na tela, a lista passa a ter **scroll**, permitindo visualizar todos os itens sem quebrar o layout.

### Responsividade
O layout foi adaptado para funcionar em diferentes tamanhos de tela.

⚠️ A responsividade ainda pode precisar de alguns ajustes futuros.
